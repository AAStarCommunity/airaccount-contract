# Funding & Transparency

AirAccount is an open-source, non-commercial smart wallet developed as part of academic research at Carnegie Mellon University.

## Funding Sources

| Source | Type | Status |
|--------|------|--------|
| CMU PhD Research Grant | Academic | Active |
| AAStar Community Open Source Initiative | Community | Active |
| GitHub Sponsors | Individual | [Open](https://github.com/sponsors/jhfnetboy) |

## Principles

- **No VC funding**: AirAccount has not accepted venture capital investment
- **No token sale**: No governance token or token sale planned for the contract layer
- **GPL-3.0**: All code is permanently open source under GPL-3.0
- **Academic origin**: Developed as part of research into ERC-4337 account abstraction UX

## Grant Applications

We actively apply for ecosystem grants to fund continued development and security audits:

- Ethereum Foundation (ESP) — applied for security audit funding
- OP Stack Retroactive Public Goods Funding — applied
- Arbitrum Foundation Grants — planned

## Security Audit Funding

The primary use of any grant funding is:
1. Professional security audit (CodeHawks/Code4rena — target $15–20k prize pool)
2. Immunefi bug bounty program setup (~$50k initial vault)
3. Multi-chain deployment gas costs
